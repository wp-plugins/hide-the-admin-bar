== Description ==

Hides the admin bar on your blog view.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the folder `hide-the-admin-bar` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= What does it do? =

It will hide the wordpress admin bar from your blog view.

= Why would i want to use it? =

In some designs it doesnt look nice or you don't need to use it.
